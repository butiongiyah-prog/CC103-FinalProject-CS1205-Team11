## 🛠️ Data Structures & Contributions

### 📚 Stack Implementation (LIFO)
**Developed by: Beverly Rose T. Buendia**

In this project, I implemented a Stack data structure to handle the Undo History feature. 
* **Functionality:** Every time a product is added or sold, the action is "pushed" onto the stack.
* **Undo Logic:** When the user chooses to undo, the system "pops" the most recent action to revert the inventory state.
* **Code highlight:** Includes custom `push()`, `pop()`, and `viewUndoStack()` methods.
